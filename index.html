웹 호스팅


인터넷에 연결된 컴퓨터 하나하나를 호스트라고 한다.
그리고 웹 서버를 운영하기 위해 호스트를 빌려주는 회사를 웹 호스팅 업체라고 한다.
우리는 무료 웹 호스팅 업체를 이용할 예정이다.

깃허브 github

깃허브 주소 : https://github.com

깃허브 웹 호스팅 절차

  1. 회원 가입
  2. new repository (저장소)
  3. 저장소 이름
  4. initialize this repository with
  5. Aad a readme file(이것만 체크 하도록 하세요)
  6. upload file
    7. 생성한 html 파일을 올리도록 하세요.
  7. setting--->pages

    8. souce - branch main your site is to be published at : 웹 페이지 주소가 생성된다.
그리고 이 페이지 주소를 웹 사이트에 입력하면 입력하면 우리가 만든 페이지에 접근할 수 있게 된다.
